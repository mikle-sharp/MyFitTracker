# Шрифты

В этой папке находится шрифт Inter (SIL Open Font License).

Для использования как кастомного шрифта:
1. Переименуйте Inter-Regular.ttf в MyFont.ttf
2. Шрифт автоматически применится к приложению

Файлы:
- Inter-Regular.ttf - основной (400)
- Inter-Medium.ttf - средний (500)
- Inter-SemiBold.ttf - полужирный (600)
- Inter-Bold.ttf - жирный (700)
